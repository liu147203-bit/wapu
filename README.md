# Wapu AI Video Challenge — GitHub Pages

This package is ready for GitHub Pages.

1. Upload `index.html` and `.nojekyll` to the repository root.
2. In **Settings → Pages**, choose **Deploy from a branch**.
3. Select `main` and `/ (root)`, then Save.
4. Open `https://YOUR_GITHUB_USERNAME.github.io/YOUR_REPOSITORY/`.

Note: GitHub Pages is static hosting. The form works as a device-local demo (localStorage); it does not collect all registrations into a shared backend.
